Scaling Up Simulation Environments for Robotic Skill Learning with Foundational Generative Models

Abstract

We propose Gen2Sim a method for scaling up development of simulation environments task definitions and reward functions with minimal human involvement by capitalizing over large scale pre-trained generative models of language and images, one step towards fully autonomous robotic manipulation skill acquisition in simulation. We generate high fidelity 3D models from a single real or generated image of an object by differentiable 3D model fitting that maximizes likelihood of a pre-trained view and pose conditioned image diffusion model. Our method queries Large Language Models (LLMs) for plausible ranges of physical parameters for the generated assets, enabling reasonable simulated behavior. Our method then chain-of-thought prompts LLMs to parse URDF files of generated or human-developed assets, and create language task descriptions and corresponding reward functions based on the assets' affordances. We demonstrate our method effectively generates meaningful tasks and credible reward functions that can be utilized by readily available RL methods for downstream skill learning. Our work contributes thousands of simulated environments and task variations for articulated and novel 3D object assets, establishing a fully automated pipeline that supports continual simulation development for robotics.

```
def compute_reward(self):
    # reward function
    door_handle_pose = self.env.get_position_by_asset_link_name("microwave", "handle_0")
    gripper_pose = self.env.get_robot_gripper_position()
    distance_gripper_to_handle = torch.norm(door_handle_pose - gripper_pose, dim=-1)
    door_state = self.env.get_state_by_asset_joint_name("microwave", "joint_0")
    cost = distance_gripper_to_handle - door_state
    reward = - cost

    # success condition
    target_door_state = self.env.get_joint_upper_limit("microwave", "joint_0")
    success = torch.abs(door_state - target_door_state) < 0.1

    return reward, success
```
